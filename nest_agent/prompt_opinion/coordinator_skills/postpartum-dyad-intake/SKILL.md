# Postpartum dyad intake

## When this skill applies

Use when the session concerns **postpartum recovery**, **newborn care in the first weeks**, **discharge readiness**, or **care coordination** for a birthing parent and/or their infant.

## Behavior

- Treat **mother and infant as two linked patients**; do not collapse infant-risk topics into “parent only.”
- Ask for or confirm: **postpartum day (PPD)**, **delivery mode**, **blood pressure** and hypertensive history, **bleeding or pain**, **EPDS / PHQ-9** (and suicide-risk items when documented), **feeding**, **weight trend**, **TSB/bilirubin** if jaundice is relevant, **medications**, and **practical barriers** (coverage, food, housing, transport, support).
- If the user reports **severe-range BP**, **stroke-like symptoms**, **heavy bleeding**, **suicidal ideation**, **respiratory distress in newborn**, **fever in newborn**, or **refusal to feed**, respond with **immediate escalation guidance** appropriate to the setting (e.g., emergency services / OB / pediatrics per local protocol) while staying within the product’s safety policy.

## Handoff

When a full **evidence-linked transition plan** (score, timeline, med card, dual red flags, task board, audit log) is needed, **consult the external NEST agent** rather than improvising that artifact inline.
