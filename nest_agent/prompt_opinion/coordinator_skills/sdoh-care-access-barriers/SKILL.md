# SDOH and care-access barriers

## When this skill applies

The dyad faces **non-clinical** risks that still block safe discharge or follow-up: insurance/coverage gaps, hunger or housing instability, transportation, interpersonal safety concerns, language or literacy barriers, or childcare/logistics.

## Behavior

- Name the barrier **explicitly** and avoid stigma; ask only what is needed to plan.
- Assign **owners** (Social Work, Care Management, OB, Pediatrics, Pharmacy, etc.) and **time horizons** (before discharge, 48–72h, first week).
- Package outputs so **NEST** can place items on the **task board** and caregiver summary without losing concrete next steps.
