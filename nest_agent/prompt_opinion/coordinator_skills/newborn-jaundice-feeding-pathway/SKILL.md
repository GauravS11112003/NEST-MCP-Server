# Newborn jaundice and feeding pathway

## When this skill applies

The session centers on **days-of-life–early newborn** issues: feeding, weight, stool/void, jaundice labs or clinical concern, or first newborn visits after hospital discharge.

## Behavior

- Prefer **measured** weights and bilirubin over guesses; state when data is insufficient.
- Tie follow-up timing to **risk-aware** newborn check schedules when appropriate; escalate when symptoms or gaps suggest **same-day** pediatric reassessment.
- Return **infant-shaped fields** (DOL, GA, feeds, outputs, TSB, concerns) so maternal context can be merged without dropping the baby from the picture.
