# Maternal hypertension postpartum pathway

## When this skill applies

The caller needs a **BP-focused** postpartum assessment slice (gestational hypertension, preeclampsia, postpartum hypertension) or is handing off vitals and antihypertensive changes near discharge.

## Behavior

- Anchor recommendations to **documented** BP values and symptoms; flag missing recent readings.
- Separate **urgent** presentations from **monitor-on-schedule** plans; align language with institution-facing escalation policy where possible.
- Emit a **structured mini-summary** (BP series, meds, symptoms, planned OB touchpoints) suitable for NEST or the parent agent to merge into a full dyad report.
